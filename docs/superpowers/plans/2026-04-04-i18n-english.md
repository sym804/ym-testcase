# i18n (English) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add English language support to the entire frontend (UI + manuals) using react-i18next, with a KO/EN toggle in the Header.

**Architecture:** react-i18next with namespace-based JSON locale files. Each page/component gets its own namespace. Language preference stored in localStorage. AG Grid locale dynamically switches via i18n language change listener.

**Tech Stack:** react-i18next, i18next, i18next-browser-languagedetector

---

### Task 1: Install Dependencies & Create i18n Config

**Files:**
- Modify: `frontend/package.json`
- Create: `frontend/src/i18n/index.ts`

```bash
cd frontend && npm install react-i18next i18next
```

- [ ] **Step 2: Create i18n initialization file**

Create `frontend/src/i18n/index.ts`:

```ts
import i18n from "i18next";
import { initReactI18next } from "react-i18next";

import koCommon from "./ko/common.json";
import koLogin from "./ko/login.json";
import koRegister from "./ko/register.json";
import koHeader from "./ko/header.json";
import koProject from "./ko/project.json";
import koTestcase from "./ko/testcase.json";
import koTestrun from "./ko/testrun.json";
import koDashboard from "./ko/dashboard.json";
import koSettings from "./ko/settings.json";
import koReport from "./ko/report.json";
import koCompare from "./ko/compare.json";
import koAdmin from "./ko/admin.json";
import koManual from "./ko/manual.json";
import koAdminManual from "./ko/adminManual.json";

import enCommon from "./en/common.json";
import enLogin from "./en/login.json";
import enRegister from "./en/register.json";
import enHeader from "./en/header.json";
import enProject from "./en/project.json";
import enTestcase from "./en/testcase.json";
import enTestrun from "./en/testrun.json";
import enDashboard from "./en/dashboard.json";
import enSettings from "./en/settings.json";
import enReport from "./en/report.json";
import enCompare from "./en/compare.json";
import enAdmin from "./en/admin.json";
import enManual from "./en/manual.json";
import enAdminManual from "./en/adminManual.json";

const savedLang = localStorage.getItem("lang") || "ko";

i18n.use(initReactI18next).init({
  resources: {
    ko: {
      common: koCommon,
      login: koLogin,
      register: koRegister,
      header: koHeader,
      project: koProject,
      testcase: koTestcase,
      testrun: koTestrun,
      dashboard: koDashboard,
      settings: koSettings,
      report: koReport,
      compare: koCompare,
      admin: koAdmin,
      manual: koManual,
      adminManual: koAdminManual,
    },
    en: {
      common: enCommon,
      login: enLogin,
      register: enRegister,
      header: enHeader,
      project: enProject,
      testcase: enTestcase,
      testrun: enTestrun,
      dashboard: enDashboard,
      settings: enSettings,
      report: enReport,
      compare: enCompare,
      admin: enAdmin,
      manual: enManual,
      adminManual: enAdminManual,
    },
  },
  lng: savedLang,
  fallbackLng: "ko",
  defaultNS: "common",
  interpolation: { escapeValue: false },
});

export default i18n;
```
